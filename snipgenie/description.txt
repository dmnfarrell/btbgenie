Background
----------

This is a Python package for doing variant calling and making phylogenies using
WGS data from microbial species.

Dependencies
------------

numpy
pandas
biopython
matplotlib
future
pyside2 (for gui only)

Installation
------------

  clone the repo and run setup.py

Usage
-----

  pathogenie
